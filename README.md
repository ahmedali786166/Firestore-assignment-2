# 🚀 User Management System

Firebase Authentication aur Firestore Database ka istemal karte hue banaya gaya ek clean aur responsive **User Management System** (with Premium Glassmorphic UI).

## ✨ Features
1. **Authentication:** Secure Email/Password Signup & Login functions along with persistent state protection (`onAuthStateChanged`).
2. **User Data Form:** Real-time collection mapping (Full Name, Email, Age, City, Profession).
3. **Reactive Users Table:** Live directory stream using Firestore `onSnapshot` data streaming.
4. **CRUD Actions:** Complete inline implementations for Single Record View, Document Update (Edit), and Document Purging (Delete).
5. **Modern Glassmorphic UI:** Built fully custom using CSS variables, custom toast alerts, and modal states.

---

## 🛠️ Project Structure
* `index.html` — Authentication Gate (Login / Signup)
* `dashboard.html` — Live Directory & Form Dashboard
* `firebase-config.js` — Firebase SDK Core Configuration
* `auth.js` — Authentication Handler
* `app.js` — Main CRUD Data Processor
* `style.css` — Core Aesthetics & Responsive Layouts

---

## ⚡ How to Set Up
1. Project clone karein.
2. `firebase-config.js` file khol kar usme apni Firebase Project credentials (`apiKey`, `authDomain`, etc.) configure karein.
3. Kisi bhi local engine ya VS Code **Live Server** extension ke sath run karein!
